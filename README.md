# Pesquisa de Satisfação - TudoWeb

Programa simples em Python que coleta a opinião de 50 clientes sobre o atendimento.

## Como funciona

1. Para cada um dos 50 entrevistados (exemplo), o programa solicita:
   - Nome
   - Idade
   - Opinião (1 = EXCELENTE, 2 = BOM, 3 = RUIM)

2. Ao final, exibe:
   - Total de respostas "EXCELENTE"
   - Total de respostas "RUIM"

## Como executar

Certifique-se de ter Python instalado (versão 3.x). No terminal, execute:

```bash
python EdipoHenrique_Ag8_DS_I.py
